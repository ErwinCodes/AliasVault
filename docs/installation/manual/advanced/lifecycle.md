# Stopping and starting AliasVault
You can stop and start AliasVault via the default docker compose commands.

## Stop
To stop AliasVault:
```bash
docker compose down
```

## Start
To start AliasVault:

```bash
docker compose up -d
```

## Restart
To restart AliasVault:

```bash
docker compose restart
```
